Regression Models

Setup: The results column was converted from W/L to binary outcomes 1/0. Movement and Team columns were dropped. Target column Next Week Rank was split off. Train/test split set to .75/.25.

Post Week 4: The Multiple Linear Regression Model is performing the best, with an R2 of approximately .933. Scaling values prior to fitting had little effect on the model. Random Forest and AdaBoost Regressions had R2 in the .80s.

Post Week 5: Linear Regression Model improved to .941. Random Forest and AdaBoost Models at near or above .90.

Post Week 6: Slight decline in R2 of Linear Models and AdaBoost (change < .01), small decrease in accuracy of Decision Tree, Random Forest, and Scaled AdaBoost (~.03).

Post Week 7: Positive change in R2 for all Linear Models, with Random Forest models improving the most (+.102).

Classification Models

Movement Predictor

Setup: Same as for regression except target column was set to values [0, 3] where 0 stands for "dropped out of top 25", 1 for "moved up in top 25", 2 for "moved down in top 25", and 3 for "stayed at same place in top 25".

Post Week 4: Results were overall unimpressive with best accuracy around .71 for SVM classifier.

Post Week 5: Random Forest Classifier improved to accuracy of .77, SVM model accuracy declined.

Post Week 6: Minor increase in accuracy of Decision Tree and SVM Model. Concerning that the Decision Tree has had the same score as the SVE Model for the last two weeks. Minor decrease in accuracy for other models.

Post Week 7: Decrease in accuracy of all models up to -.219 (Unscaled SVM).

Bin Predictor

Setup: Same as for regression except rank feature and target column were set to values [0, 5] where 0 stood for ranks 1-5, 1 for ranks 6-10, etc., and 5 for unranked.

Post Week 4: Results were worse than for Movement Predictor. Top accuracy score for Decision Tree Classifier, with accuracy around .46.

Post Week 5: Significant improvement in Random Forest Model to accuracy of .645.

Post Week 6: Minor decrease in accuracy across all models. Unscaled SVM had worst decline of .105.

Post Week 7: Improvements in all models ranging from +.012 (Scaled SVM) to +.149 (Decision Tree).
